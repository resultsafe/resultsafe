# @resultsafe/core-fp-shared — Package Context

<!-- AGENT: update only when this package's purpose or scope changes -->

## What this package does

Provides shared utilities, types, and constants used across all functional programming packages.
This package contains common functionality that should not be duplicated across
individual FP implementations.

---

## Why it exists as a separate package

This package isolates shared FP utilities to prevent circular dependencies
and code duplication across the resultsafe monorepo packages.
