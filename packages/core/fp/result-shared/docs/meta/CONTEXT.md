## What this package does

Provides shared utilities, types, and constants used by Result-related packages.
This package contains common functionality that should not be duplicated across
Result implementations.

---

## Why it exists as a separate package

This package isolates shared Result utilities to prevent circular dependencies
and code duplication between Result-related packages.
