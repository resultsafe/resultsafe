# @resultsafe/core-fp-union — Package Context

<!-- AGENT: update only when this package's purpose or scope changes -->

## What this package does

Provides utilities for working with tagged unions (sum types) in TypeScript.
Enables pattern matching and type-safe handling of union types in functional style.

---

## Why it exists as a separate package

This package isolates union type utilities, providing a foundation for types like
Result and Option which are implemented as tagged unions.
