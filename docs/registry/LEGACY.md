## How to add an entry

When deprecating something:
1. Add a row to the appropriate table above
2. Add `> ⚠️ DEPRECATED: [date]. Current: [link]` to the deprecated file/symbol
3. Update CHANGELOG.md with the deprecation
4. Set a removal date (usually next major version)
