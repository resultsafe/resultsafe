## рџљЂ Next Steps for Full Deployment

### Step 1: Fix Broken Links (Optional for dev)

The build fails due to broken links check. For development, you can skip this:

```bash
# Option A: Disable strict link checking
# Edit docusaurus.config.ts:
onBrokenLinks: 'warn',  // instead of 'throw'
onBrokenMarkdownLinks: 'warn',
```

### Step 2: Generate Translations

```bash
cd docs
pnpm run write-translations
```

This creates translation files in `i18n/en/` and `i18n/ru/`.

### Step 3: Start Development Server

```bash
cd docs
pnpm run start
# Opens at http://localhost:3000
```

### Step 4: Build for Production

```bash
cd docs
pnpm run build
# Output: docs/build/
```

### Step 5: Deploy to GitHub Pages

```bash
cd docs
pnpm run deploy
```

Or manually:
```bash
# Build
pnpm run build

# Deploy using gh-pages
npx gh-pages -d build
```
