# Документация на русском языке

Русскоязычный перевод canonical документации проекта resultsafe.

> **Важно:** эти документы являются переводом.
> Canonical источник истины - английская документация в `docs/`.
> При расхождении между переводом и оригиналом приоритет у оригинала.

## Статусы переводов

| Статус   | Значение                                  |
| -------- | ----------------------------------------- |
| actual   | перевод актуален, соответствует оригиналу |
| outdated | оригинал изменился, перевод устарел       |
| draft    | перевод в процессе                        |
